# GJWNetwork
基于AFNetworing封装的下载库，支持自动配置请求头，支持GET/POST上传下载功能

## version 0.1.3
* 基于AFNetworing3.0.4
* 支持iOS7.0及以上版本
* 支持pod： pod 'GJWNetwork', '~> 0.1.3'

## 集成说明
你可以在Podfile中添加一行代码来使用GJWNetwork
```
pod 'GJWNetwork', '~> 0.1.3'
```
你也可以手动添加源码使用本项目，导入AFNetworing3.0.4及以上版本
